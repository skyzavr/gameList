# About

A game list app made with React.js, TS, RTK & CSS Modules

Feautures:
- Searching
- Filtering
- Sorting
- Routing
- Light/Dark theme
- Card's lazy loading

### Stack

- React
- Typescript
- Vite
- RTK (a bit)
